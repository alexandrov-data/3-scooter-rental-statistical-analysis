# Статистический анализ сервиса аренды самокатов

## О проекте
В проекте исследуются данные сервиса аренды самокатов, чтобы понять, как пользователи разных тарифов используют сервис и какие различия между ними значимы с точки зрения бизнеса.

Цель — проанализировать поведение пользователей, сравнить тарифы и проверить гипотезы, которые помогают принимать решения о развитии продукта и тарифной политики.

## Бизнес-задача
Сервису важно понимать, чем отличаются пользователи с подпиской и без подписки, как они влияют на выручку и какие продуктовые решения могут повысить эффективность бизнеса.

Анализ помогает ответить на ключевые бизнес-вопросы:
- как различается поведение пользователей разных тарифов;
- какие группы пользователей приносят больше ценности;
- есть ли статистически значимые различия между сегментами;
- какие выводы можно использовать для развития подписки и роста выручки.

## Задачи
- подготовить и проверить исходные данные;
- провести исследовательский анализ данных;
- изучить поведение пользователей по тарифам;
- рассчитать основные показатели, связанные с поездками и выручкой;
- проверить статистические гипотезы;
- интерпретировать результаты и сформулировать выводы для бизнеса.

  ## Задачи
- подготовить данные для анализа: загрузить таблицы, проверить структуру, привести названия столбцов и типы данных к корректному виду;
- выполнить очистку данных: найти и обработать пропуски, дубликаты и возможные аномалии;
- объединить данные о пользователях, поездках и тарифах в единую таблицу;
- провести исследовательский анализ данных и изучить основные характеристики пользователей и поездок;
- сравнить пользователей разных тарифов по длительности поездок, расстоянию и интенсивности использования сервиса;
- рассчитать показатели, влияющие на выручку сервиса, и оценить различия между тарифными группами;
- проверить статистические гипотезы о поведении пользователей и значимости различий между сегментами;
- подготовить аналитические выводы и рекомендации, которые могут быть использованы для развития тарифной модели и повышения эффективности сервиса.
